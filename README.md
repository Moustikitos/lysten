# lysten
The lightest way to bridge something with ARK


## Set a trigger

```python
from lysten import core
core.setSenderIdTrigger("DKf1RUGCM3G3DxdE7V7DW7SFJ4Afmvb4YU", "^arky *(\d*)[^\d].*", "test")
```

## Parse blocks

```python
from lysten import core
core.main()
```
